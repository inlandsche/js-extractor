# JS Extractor - Walkthrough

## Overview
The **JS Extractor** is a cyber security tool designed to scan JavaScript files and text for sensitive information like API keys, secrets, URLs, and paths. It features a nostalgic **Super Mario Bros.** aesthetic.

## üöÄ Getting Started

### Option 1: Docker (Recommended)
1. **Prerequisites**: Install Docker and Docker Compose.
2. **Start**:
   Open a terminal in the project root and run:
   ```bash
   docker-compose up --build
   ```
   > App runs on `http://localhost:5173`  
   > Backend API on `http://localhost:3001`

### Option 2: Manual Setup
1. **Prerequisites**: Node.js installed on your system.
2. **Start the Backend**:
   ```bash
   cd backend
   npm install
   node server.js
   ```
3. **Start the Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

## üõ°Ô∏è Usage Features

### Analyze Text (Paste Code)
1. Click the **PASTE CODE** button (styled like a pipe/brick).
2. Paste any JavaScript code into the text area.
   *Example:* `const apiKey = "AIzaSyD-FakeKey";`
3. Click **START LEVEL!**.
4. View the results in the log below.

### Analyze File (Question Block)
1. Click the **UPLOAD FILE** button.
2. Click the **? Block** to select a file.
3. Click **START LEVEL!**.

### Interpreting Results
- **LEVEL COMPLETE**: Analysis finished successfully.
- **PRINCESS IS IN ANOTHER CASTLE**: No secrets found.
- **GAME OVER**: Error occurred.
